### 数の暴力

とりあえず、私はそこまで頭が良くないので serf のクラスタリングのさわりだけやってみます。

32GBの実装メモリがあるので、これを可能な限り活用してみようと思います。  
とりあえず必要最小限のアプリだけ立ち上げてメモリの使用量を減らしまくります。

![メモリ使用量](memory_usage.png)

1インスタンス512MBの割り当てとして立ち上げた場合

```
(29 - 2) * 1000 / 512 = 52
```

52台分の仮想マシンが構成できます。

もっと頑張れば

```
(29 - 0.5) * 1000 / 512 ≒ 56
```

いける。
